Event Hub Sourcetype Detail

The Splunk Add-on for Microsoft Cloud Servies includes pre-trained sourcetypes for certain data sources.

  • azure:monitor:aad - used for Microsoft Entra ID (Azure Active Directory) events such as sign-in events and Azure AD audit events.
  • azure:monitor:resource - used for Azure resources such as Cosmos DB and Azure Data Share.
  • mscs:azure:eventhub - used for generic event hub events. This sourcetype does not have any CIM mappings out-of-the-box.
